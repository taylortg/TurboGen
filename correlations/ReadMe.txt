These models in the correlations directory are used for preliminary sizing of an impeller.
They are based on empirical test data curve fits, so they are not meant to be robust. 
Instead think of them as a guide for if a certain duty can be achieved by an impeller at your given constraints.

Correlations based on text from:
1. Casey and Robinson (2021)
2. Aungier (2000)
3. Whitfield (1990)
